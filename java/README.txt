Java Coursework Pack
====================

Main Java file to run:
- src/LibraryFxApp.java

Core Java files included:
- src/Book.java
- src/Student.java
- src/Transaction.java
- src/Validator.java
- src/LibraryFxApp.java

Optional reference file:
- optional/LibraryConsoleBackup.java
  This is NOT required for the final JavaFX app. It is kept only as a safety/reference copy.

Shared data repository files:
- data/book.csv
- data/student.csv
- data/transactions.csv

JavaFX IntelliJ VM options example:
--module-path "C:\Users\User\Documents\FX\javafx-sdk-25.0.2\lib" --add-modules javafx.controls --enable-native-access=javafx.graphics

Notes:
- This pack is kept at a first-year level: classes, validation, file import, reports, search, export.
- If you already have your own shared CSV data from the Python part, you can replace the sample data files in the data folder.
- If you do not want the optional console backup, you can ignore or delete the optional folder.
